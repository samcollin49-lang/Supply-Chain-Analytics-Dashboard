Supply Chain Analytics Dashboard:

This project analyzes the DataCo Global Supply Chain dataset and presents key insights through an interactive Streamlit dashboard, a written Overleaf report, and a Beamer presentation.

What We Did

Cleaned and prepared the dataset

Built KPIs such as delivery delays, profitability, and late-delivery rates

Created visualizations for category profit, shipping delays, and country-level performance

Developed a Streamlit dashboard for interactive exploration

Repository Contents

app.py — Main Streamlit dashboard

module1.py — Data loading & preprocessing functions

Data/ — Dataset used in the dashboard

Supply_Chain_Analytics_Dashboard_Project.pdf — Final project report

BEAMER_PRESENTATION.pdf — Slide presentation
